无线设置

1. 无线基础设置
配置路由器无线网络的基本参数。
无线状态 显示当前的无线工作状态 。
无线开关 用来开启或者关闭无线功能.
网络模式 一般采用默认b/g/n模式即可
网络名称(SSID) 主SSID，这是必填的
SSID1-7 多SSID，你可以启用多个的SSID。
广播网络名称(SSID) 是否广播SSID，禁止广播后，无线客户端将不能收索的本设备的SSID
AP Isolation 本AP内隔离，启用后本AP内的客户端不能互相访问，可以防止病毒的传播。
MBSSID AP Isolation 本AP外隔离，不属于本AP的其他客户端不能访问本AP下面的客户端。
频段 无线工作时使用的频率，如果周围有别的无线网络建议将无线频段的范围设置开一些，避免太多的干扰


2. 无线安全设置
安全配置: 设置无线网络的安全性。不同的SSID可以选择不同的加密方法，根据需要选择不同安全模式和加密类型，够阻止未授权用户对无线网的访问。
SSID选择 选择您需要加密的SSID
安全模式 选择需要的安全模式
加密类型 选择需要的加密类型

3. 无线访问策略
访问控制: 提供了对无线访问策略的设置,可以设置允许和拒绝所选的MAC地址的接入，此功能可以配合不同的SSID使用。
SSID选择 选择您需要设置访问的SSID，全部一起不超过四组。
功能 您可以选择所采用的策略，在同一个SSID下的所有增加条目只能采用其中的一个策略。
新增 填入您所需增加的条目，格式如：00:ed:23:da:eb:18

4. 无线分布系统(WDS设置)
无线分布系统: 无线分布系统借助无线媒介像以太网一样同其它的AP通讯。
WDS 模式 1. 懒人模式：开启懒人模式后，系统会自动发送自己的匹配信息给其它AP而不需要填入对方MAC地址，但另一方需要填入本设备的无线MAC地址。
2. 桥接模式：需互相输入对方MAC地址，一般在桥接模式下使用。
3. 中继模式：使用方法同桥接模式，一般在路由模式下使用
Phy 模式: 一般采用默认的CCK模式即可
加密方式: 选择采用的加密方式，也可以选择NONE表示不加密
密钥: 当有加密方式时，必须是相同的加密方式加相同的密钥，两AP之间才能互相通讯
AP MAC 地址 填入您所需要连接的MAC地址 ，格式如 00:ed:23:da:eb:18

5. WPS设置
WPS设置: 可以使用无线网卡更加方便，简单的连入我们的路由器，它可以穿透我们设置的安全方法和加密类型进行连接。 有两种方式可供选择(请先把WPS设置为启用)：
1. PBC(按键)方式

有两种方法可以启用PBC方式，
1. 在配置界面上选择PBC，然后点击提交按钮。
2. 按一下我们设备上的WPS按钮
启动成功后：WPS灯会不断闪烁，这时只需要在无线网卡上点击PBC按钮就可以成功连接了

2. PIN码方式 在配置界面上选择PIN，然后再输入无线网卡客户端的PIN码，然后点击提交。
再在相应的无线网卡界面进行连接就可以了。

6. 无线高级设置
高级配置: 对无线高级参数的设置,一般情况下并不需要对这些作修改,保持系统默认的就可以了.如果您对无线参数非常了解,可以按您的需求进行设置。

7. WIFI 多媒体
WIFI多媒体: 提供了对无线多媒体参数的设置,wmm 允许无线通信根据数据类型定义一个优先级范围。时间敏感的数据，如视频/音频数据将比普通的数据有更高的优先级。为了使wmm 功能工作，无线客户端必须也支持wmm。客户可以根据需求选择是或否。

WAN 设置

WAN是广域网(Wide Area Network)的缩写。通过WAN的设置可以让您正确的连入互联网(Internet),首先选择所需要的WAN口连接类型,我们提供了五种接入互联网的方式供您选择，分别是比较常用的：静态IP，动态IP，PPPOE和不常用的PPTP，L2TP方式。您必须选择其中的一种接入方式，通常ISP(Internet 提供商)会告诉您所采用的接入方式(选择正确的连接方式非常重要，这关系到您是否能正常上网)。如果您不清楚的话，可以向您的ISP咨询，也可以用我们的自动侦测功能进行侦测。路由器默认的是静态IP方式。
这里需要注意的是：当您点击自动侦测按钮进行自动侦测后，请务必不要关闭本页面直到检测结果出来，否则会造成假死机状态，整个检测过程不会超过两分钟，请耐心等候！

网络安全

1.防火墙:
此功能用于用来对路由器的防火墙进行调整的,一般情况下,您只需要保持默认配置就可以了.如果您想要提高系统的安全性，并通过状态来判断数据包的允许与拒绝,可以启用它.但这样也有可能导致您的一些既定策略(如IP/MAC/PORT过滤)失效，请慎重使用。启用后请重新启动路由器一遍。。

2. MAC/IP/PORT过滤：
此功能用于路由器所带的客户端进行限制管理的.如果你需要限制路由器所带机器的上网情况,可以使用本功能.要使用本功能首先您得启用它并且选择一种默认策略(接受或者拒绝),表示不符合规则的数据包被接受或者拒绝.然后填写相应的过滤规则，注意并不是所有项都要填满，而是根据您自己的需要填写相应选型。例如，要禁止IP为192.168.1.146的IP 上网(其它都可以上网)，只需要使用默认策略为接受的策略，并在源IP地址一栏里填入192.168.1.146 就可以了. 运用本功能，能够提高用户局域网的安全性和可管理性。

3. 域名过滤
用于管理和限制局域网用户访问互连网. 阻止局域网用户访问包含特定URL关键值的网站的。如果您不想让局域网的机器访问某个网站，可以使用此功能。如填入sohu.com ，这样不仅www.sohu.com不能访问，其它的sohu.com的二级或者三级域名也将不能访问。。

4.主机过滤
用于管理和限制局域网用户访问互连网. 阻止局域网用户访问包含特定主机的网站的. 如填入sports ，这样所有关于sports的URL将不能访问，如sports.163.com,sports.sina.com.cn等都不能访问.注意：不支持中文。

5. 静态路由
用来设置路由器静态路由表的，如果您需要为特定的主机增加特定的路由，可以使用本功能。注意：系统自有的路由表是删除不了的，您只可以删除自己设置的路由表条目。


系统服务

1. DMZ设置:
当您想把局域网内的某台主机开放到互联网的时候，可以使用本功能，这样，互联网上的其它机器访问路由器WAN口IP就相当于访问了您设置的那台局域网的主机。注意：当这样设置后，那台局域主机将暴露在互联网上，不再受到路由器的保护，请慎重使用。

2. 虚拟主机：
设置虚拟服务器 ：可以把发往WAN口的数据包转发到局域网内的某台主机的某个端口上。

3. WEB远程管理
用于设置web远程管理，如果开启远程管理的功能，将允许外网访问Web配置页面 .但是为了安全着想一般情况下请关闭它。

4. 动态域名服务(DDNS)
所谓动态域名服务，是服务提供商提供给用户的一种服务，将一个固定有效的互联网域名(URL)同用户的设备绑定.

5. NTP设置
对路由器自己的时间进行设置，您可以点击"和本机同步"按钮让路由器时间和本台PC时间同步，也可以填入时间服务器从internet上进行时间更新。.


系统管理

1. 系统模式:
提供两种模式供您选择，桥接模式和路由模式。路由模式为默认的工作模式，本设备一般工作在此模式下；当您想把本设备作为一个纯AP来使用的话，可以选择桥接模式，此时，所有以太网口(含LAN和WAN)和无线口将连成一个口，所以基于WAN口的功能将会无效(如防火墙，拨号等功能等)。

2. 恢复出厂设置:
如果您想把系统恢复到出厂时候的状态，可以使用此功能，一般在您把系统设的很混乱，想重新进行设置的时候使用。注意：使用本功能后您自己的设置将会被清空，另外系统会在恢复完成后重新启动一次，大概会需要一分钟左右时间，请耐心等候。。

3. 固件升级：
用户对固件进行升级。您可以在我们的网站（www.kingnet.com.cn）上下载对应产品和型号的新软件进行升级。注意：请务必确认您下载的升级软件是匹配的，否则会造成系统损坏。另外在升级过程中请务必保持您的PC与路由器的连通和不要关闭电源，否则容易造成系统损坏，系统升级成功后，会出现提示信息，请耐心等候。

4. 保存/加载配置文件
如果您的配置比较多，为了方便下一次的配置，可以使用保存配置文件功能把当前您的配置信息保存起来，需要设置的时候，把该配置文件加载进系统就可以了，节省了时间。 加载配置文件功能用来导入您自己保存的配置文件，系统会根据您的配置文件更新当前的系统配置。注意：这两个功能一般是连用的，除非您非常了解，否则请不要改动保存下来的配置文件后进行加载

5. 系统命令
用来运行一些系统自带的小程序，如ifconfig,等等。

6. 密码管理
用来设置系统的登录用户名及其密码，为了安全着想，请您务必改变默认的密码，否则可能会造成其它人员采用默认密码登录路由器进行破坏。

7. 重启

LAN设置

1. LAN 设置
设置LAN(本地局域网)参数：一般情况下您只需要设置其中的：IP地址，子网掩码以及DHCP的地址池就可以了。IP地址为路由器设备的地址，如果您修改了它，那下次登陆就必须用修改后的IP地址进行登陆,并且为了保证DHCP服务器正常工作,请同时修改DHCP的地址池同LAN IP地址在同一个网段。DHCP方式让您选择是否开启路由器的DHCP服务器，如果您选择开启的话，那路由器下面的PC就可以通过自动获取得到IP地址了。注意您的LAN IP地址不能和WAN口的IP地址在同一个网段，否则会导致系统异常。
IP地址 设备的LAN口的地址，局域网的计算机可以通过该IP地址管理路由器。缺省为 192.168.1.1 。
子网掩码设备的子网掩码为 255.255.255.0 。
DHCP方式 禁用和启用方式，表示是否开启自动分配地址功能，默认为开启
DHCP开始IP DHCP服务器分配IP的开始地址，默认为 192.168.1.100
DHCP结束IP DHCP服务器分配IP的结束地址，默认为 192.168.1.200
更多选项 一般不常用，保持默认配置即可
1. LAN2 IP配置： 可以为路由器配置LAN口配置第二个IP地址.
2. 802.1d:生成树： 主要用来防止网络出现回路.
3. LLTD： 如有LLTP客户端，则可以自动显示路由器信息
4. UPNP：路由器可以自动转发某些应用程序需要转发的端口，通常用于可以使一些下载软件和游戏软件加快速度。
5. 路由器通过：一般用于IPV6

DHCP设置
对DHCP服务器的一些其它参数进行设置,一般情况下并不需要对这些作修改,保持系统默认的就可以了.但如果您需要一些特殊的设置,可能就会用得上,如DNS服务器同网关不是本路由器,需要加大租约时间等.另外如果您需要给某台PC分配固定的IP地址,可以使用静态设定功能.
DHCP 子网掩码 DHCP服务器分配给客户端的子网掩码,一般和LAN口的子网掩码相同.默认为 255.255.255.0
DHCP 主 DNS: DHCP服务器分配给客户端的主DNS服务器地址,一般为LAN口的地址或者真实的DNS地址,.默认为192.168.1.1
DHCP次DNS DHCP服务器分配给客户端的次DNS服务器地址,如没有的话,可以不填
DHCP默认网关 DHCP服务器分配给客户端的网关地址,一般和LAN口的IP地址相同.默认为192.168.1.1
DHCP租约时间 DHCP服务器分配给客户端的IP地址的到期时间
静态设定 如果您需要把固定的IP地址分配给固定的局域网内某台主机(以MAC地址作为标识)，可以填写相关参数.



DHCP客户表
显示通过DHCP方式连接到本路由器的客户端的相关信息，注意并不是显示所有连接到本路由器的主机，而仅是显示通过DHCP方式连到本路由器的主机.
MAC 地址 客户端的MAC地址
IP 地址: 客户端分配到的IP地址
时间 客户端还能使用该IP的时间


忘记密码

如果您没有修改过WEB管理的登录密码，那么请使用"admin"登录，注意区分大小写。

如果您曾经修改过密码而又忘记了密码，按照下面的步骤可以把密码恢复为缺省的"admin"，但是同时，在本设备上的所有设置的信息也将全部丢失，您必须重新设置所有的参数。
1. 接上设备的电源。
2. 长时间按住<Reset>按钮（至少5秒）。
3. 松开按钮，等待设备重启完成。
4. 这时候，设备已经恢复为缺省配置，网关IP为192.168.1.1，掩码为255.255.255.0，用户名和密码均为"admin"。





保存加载配置文件

备份系统设置信息
单击<导出>按钮，可以把所有的设置信息打包成一个文件，备份到您的计算机上。您以后可以使用这个文件恢复本设备的设置信息。
注意：这个文件无法用文字处理软件或者表格编辑软件修改。

从文件中恢复设置信息
利用上面保存的文件恢复本设备的设置信息。单击<浏览>按钮， 选择一个以前备份的文件，然后单击<导入>把文件上传到设备内。

恢复到出厂设置
恢复到出厂设置将清除本设备的所有设置信息，恢复到刚生产出来的缺省状态。执行后本设备将重新启动。在你把本设备从一个网络环境换到另一个不同的网络环境的时候比较有用。一旦您执行了本命令，那么原来的设置信息就彻底丢失了，建议您在操作之前备份设置信息，以便发生意外的时候可以恢复。
缺省的用户名： admin
缺省的密码： admin
缺省的IP地址（网关地址，用来配置管理路由器的IP地址）为：
192.168.1.1

修改密码

为了防止他人随意修改您的配置，您可以修改WEB管理的登录密码。
1. 输入用户名
2.输入您的新密码
3.重复输入一遍新密码
4. 单击<提交>按钮

注意: 密码是区分大小写的，输入的时候请注意。


远程管理

默认情况下，您只能在局域网中设置本路由器，只有开启远程管理后，您才可以在Internet中设置本路由器。

开启路由器远程管理功能，选择"允许 通过WAN口远程管理本设备"，这时您就可以在Internet中管理您的设备，您只要在浏览器地址栏中输入http://WAN IP就可以访问您的设备(把WAN IP换成您设备实际的WAN口IP地址)。


升级

升级功能使得您可以保持我们发布的最新版本的系统软件。您必须到我们的网站下载本设备的系统软件，保存到您的计算机的某个文件夹下面。

单击<浏览>按钮选择那个文件，然后单击<确定>按钮，文件将被上载到设备上，上载完成后，本设备将重新启动。为了防止升级过程中出现意外，建议你在升级之前保存一下设置信息。

动态域名


因为通过PPPoE或者动态获取IP地址上网时，获取到的IP地址不固定，这给想访问本局域网服务器的Internet用户带来不便。而动态域名服务器 (DDNS)解决了这个问题，设置DDNS选项后，路由器会将域名自动和更新IP地址对应，在Internet上的用户可以通过域名来访问您的局域网。
开启选择是否开启DDNS功能。
用户名您在提供DDNS服务的网站注册的用户名。
密码您在提供DDNS服务的网站注册的密码。
DNS 您在提供DDNS服务的网站注册的域名。
状态 DDNS状态。

在您使用Dynamic DNS(DDNS)功能之前，请您先到下面网站去申请注册一个动态DNS地址服务：

www.3322.org
www.dyndns.org
