DHCP Client List Help

This page shows Client Name, MAC Address, Assigned IP and Lease Time of each DHCP Client connected to this device.

  • Client Name - The name of the DHCP client.
  • MAC Address - The MAC address of the DHCP client.
  • Assigned IP - The IP address this device has allocated to the DHCP client.
  • Lease Time - The time of the DHCP client leased.

You cannot change any of the values on this page. To update this page and to show the current connected devices, click on the Refresh button.
